<a name="intro">
<p>
<b>What this list is about:</b><br>
Spammers get paid by businesses whose websites they advertise. We refuse to accept <a href="spam.htm">spam</a> from any of following domains or advertising any of these domains, as we don't want to buy from spammers. We publish this list for informational purposes only. For a URL of a plain text version of this list <a href="http://www.joewein.de/sw/bl-text.htm">see here</a>. The software that catches the spam used to build this list is available <a href="http://www.jwspamspy.com">here</a>.

<a name="how">
<p>
<b>How this list is compiled:</b><br>
Every email sent to our mailboxes is analyzed by <a href="http://www.jwspamspy.com"><b>jwSpamSpy</b></a>, our spam filter software (<a href="http://www.jwspamspy.com">free 30 day evaluation version available for download</a>). It extracts and inspects the domain names in all mail sent to us that meets a sufficient number of criteria that are typical for spam. We then perform whois-lookups (see <a href="http://www.joewein.net/bl-log/bl-log.htm">whois log</a>) and <a href="http://www.google.com">Google searches</a> on these domains, as we seek to minimize the risk of missing legitimate mail from legitimate domains. Nevertheless, nobody is perfect. If you are the owner of one of the following domains and you are not involved in spamming activity, please <a href="mailto:joewein@pobox.com">contact us</a> to get unlisted.<br>
<br>
<font size="-1">
<B>Notice:</B>
A listing here does not imply that we recommend anyone to block any mail involving these domains, only that we at <a href="http://www.joewein.de">joewein.de</a> chose to do so. If you find the following list useful, please add a link to it on your website. Thanks! 
</font><br>
